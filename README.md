# TAP CHURCH

Site institucional (MVP) em Next.js.

## Passo a passo (for dummies)

1. Instale as dependencias

```bash
npm install
```

2. Rode localmente

```bash
npm run dev
```

3. Abra no navegador

```bash
http://localhost:3000
```

## Deploy na Vercel

1. Crie uma conta na Vercel e conecte seu repositorio.
2. Clique em New Project e selecione este repo.
3. Deploy automatico.
